## Terminology
|Term|Description|
|----|-----------|
|`r8`|Any one byte CPU register|
|`r16`|Any two byte CPU register pair|
|`d8`|A constant one byte value from memory|
|`d16`|A constant two byte value from memory|
|`s8`|A constant signed 8-bit integer from memory|
